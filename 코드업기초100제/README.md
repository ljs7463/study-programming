12. 실수형(float)로 변수를 선언하고 그 변수에 실수값을 저장한 후
저장되어 있는 실수값을 출력해보자.
```
a=float(input())
print('%.6f'%a)
```

13. 정수(int) 2개를 입력받아 그대로 출력해보자.
```
a, b = input().split()
x = int(a)
y = int(b)
print(x, y)
```
split 함수를 통해 공백을 기준으로 하여 a와 b로 나누어 주어서 확실하게 정수로 받기위해서 int를 통해 
각각을 x와 y로 저장을 해주고 출력을 한다.
